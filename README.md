## ðŸ“ Syntax

### Variables
```
var x = 10
var name = "DAMAVAND"
x = x + 5
```

### Printing
```
print x
print "Hello World"
print "Age: " + x
```

### if statement
```
if x > 3 {
    print "x is big"
}
```

### While loop
```
var n = 3
while n > 0 {
    print n
    n = n - 1
}
```

### For loop
```
for i = 0; i < 5; i = i + 1 {
    print i
}
```

### Comments
```
# This is a comment
print "Hello"
```

## Example
```
var name = "World"
print "Hello " + name

for i = 0; i < 3; i = i + 1 {
    print i
}
```

```

1
2
```
